# Домашнее задание: Написание bash-скриптов

## Цель
Закрепление практических навыков по созданию и использованию скриптов на Bash в операционной системе Linux.

---

## Описание и пошаговая инструкция выполнения

### Задача №1
Создать bash-скрипт, который заменяет расширения указанных файлов на новые.

#### Входные аргументы:
- Файл (`--file`), например: `test.txt`, `test2.sh`
- Исходное расширение (`--extension`), например: `txt`
- Целевое расширение (`--replacement`), например: `sh`

#### Запуск:
bash./ext_replacement.sh --file test.txt --file test2.sh --extension txt --replacement sh


#### Ожидаемый вывод:
/home/user/test.sh
/home/user/test2.sh


---

### Задача №2
Реализовать bash-скрипт, принимающий число и вычисляющий:
- Произведение чисел первой половины диапазона от 1 до указанного числа.
- Сумму чисел второй половины диапазона.

При передаче чётного числа выполняется полный расчёт обеих частей. При передаче нечётного числа середина исключается из расчёта.

#### Примеры:
- Передано число **8**:  
  *Произведение:* $1*2*3*4=24$  
  *Сумма:* $5+6+7+8=27$

- Передано число **9**:  
  *Произведение:* $1*2*3*4=24$  
  *Сумма:* $6+7+8+9=30$

#### Запуск:
bash.
/numbers.sh 8

#### Выход:
mult: 24
sum: 27

Написать bash-скрипт, осуществляющий запуск тестов на Selenoid-сервере по указанному адресу `http://localhost:4444/wd/hub`. Скрипт принимает следующие аргументы:

### Задача №4
Создать bash-скрипт, принимающий два параметра:
- Путь к файлу (`--file`)
- Поисковую строку (`--search`)

Программа должна отображать количество найденных строк. Если совпадение отсутствует, выдаётся соответствующее уведомление.

#### Пример результата:
Не найдено ни одного совпадения в файле <полный_путь_к_файлу>
